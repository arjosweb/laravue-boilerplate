# _LaraVue - Boilerplate Start Kit_ `1.0.0`

SPA - Laravel (8) + VueJS (2)

## Get started

#### Laravel dependencies
```
composer install
```

#### Front-end dependencies
```
yarn install
```

## Run project

#### Terminal #1
```
php artisan serve
```
#### Terminal #2
```
yarn watch
```
